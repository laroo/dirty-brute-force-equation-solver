Dirty Brute Force Equation Solver
=================================

A PHP Composer class that's solves your 'complex' equations the dirty brute force way!

Example
--------------------

    ``` php
    $oSolver = new DirtyBruteForceEquationSolver();
	$fAnswerForX = $oSolver->solve('$x + ($x * $x)', 12);
    ```
